
# OOBE Automation

Automates Windows workstation setup.

## Features

- App installs

- Power settings

- Timezone sync

## Usage

Run the script from an elevated PowerShell prompt or use the included launcher:

- `run-setup-oobe.cmd`

Or from PowerShell:

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\setup-oobe.ps1
```
